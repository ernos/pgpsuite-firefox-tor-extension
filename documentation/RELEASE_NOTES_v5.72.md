# PGP Suite - v5.72 Release Notes

## Changes

### Clear Buttons Added to Input Fields

All major input areas now have a dedicated **Clear** button, making it faster to reset fields without manually selecting and deleting text.

The following tabs have been updated:

- **Keys tab — Import/Export**: A Clear button now appears below the _Import Private or Public Key_ textarea. Clicking it clears both the key input and the passphrase field.
- **Encrypt tab**: A Clear button now appears below the _Message to Encrypt_ textarea.
- **Decrypt tab**: A Clear button now appears below the _Message to Decrypt_ textarea.
- **Sign tab**: A Clear button now appears below the _Message to Sign_ textarea.
- **Verify tab**: The existing Clear button now also clears the _Message to Verify_ input in addition to resetting the verification output.
